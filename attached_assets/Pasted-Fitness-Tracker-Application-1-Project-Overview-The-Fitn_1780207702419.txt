Fitness Tracker Application
1. Project Overview

The Fitness Tracker Application helps users monitor their daily fitness activities such as steps walked, distance covered, calories burned, workout duration, BMI, and fitness goals.

Objectives
Track daily physical activity.
Monitor calories burned.
Calculate BMI.
Maintain workout history.
Generate fitness reports.

10. Resume Description

Fitness Tracker Application | Java, Swing, JDBC, MySQL

Developed a desktop-based fitness tracking system that enables users to monitor daily steps, calories burned, workout activities, BMI, and fitness goals. Implemented CRUD operations using JDBC and MySQL, designed an interactive Java Swing GUI, and generated fitness reports for activity analysis.
